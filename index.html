My Skill

Web
Programmer

Web programmer adalah suatu pekerjaan yang berfokus pada backend (Server Scripting) pada suatu perangkat lunak. Tanpa adanya backend, data yang ada pada suatu aplikasi tidak bersifat dinamis.

Linux
Sys Admin

Linux Sys Admin adalah suatu pekerjaan yang berfokus pada sisi administrasi web server. Ketika server sedang dihajar ribuan request oleh client, Sys admin akan memastikan bahwa server tidak akan down.

Mobile
Engineer (Hybrid React)

Mobile Engineer adalah suatu bidang yang berfokus pada pengembangan aplikasi mobile. Tanpa adanya mobile engineer, aplikasi mobile akan mengalami deprecated function. Hal tersebut terjadi karna jaman semakin maju, begitu juga fungsi2 didalam harus selalu diupdate agar aplikasi dapat berjalan maksimal.
